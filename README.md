# Lung Cancer Data Analysis and Machine Learning

## 📌 Project Overview

This project focuses on analyzing a lung cancer dataset and building machine learning models to classify pulmonary disease cases. It includes data preprocessing, exploratory data analysis, visualization, clustering, and supervised learning techniques.

The goal is to gain insights from the data and compare multiple machine learning algorithms to evaluate their performance.

---

## 📊 Dataset

The dataset used in this project contains medical and diagnostic features related to lung cancer and pulmonary disease classification.

---

## ⚙️ Project Workflow

### 1. Data Exploration

* Loading and inspecting the dataset
* Understanding data structure and feature types
* Checking missing values

### 2. Data Preprocessing

* Handling numerical and categorical variables
* Converting target variable into factor format

### 3. Statistical Analysis

* Descriptive statistics (mean, min, max, standard deviation, quartiles)
* Percentile and distribution analysis

### 4. Data Visualization

* Histograms for feature distributions
* Boxplots for outlier detection
* Pie chart for class distribution
* Correlation matrix visualization
* Scatterplot matrix for feature relationships

### 5. Unsupervised Learning

* K-Means clustering for pattern discovery
* Visualization of clusters for different values of K

### 6. Supervised Learning Models

The following classification models were implemented:

* Naive Bayes
* Decision Tree
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)

---

## 📈 Model Evaluation

All models were evaluated using the following metrics:

* Accuracy
* Classification Error
* Precision
* Recall
* F1 Score

The results were compared to identify the best-performing model.
